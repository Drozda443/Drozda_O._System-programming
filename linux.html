Коротка інформація про ОС GNU/Linux

Операційна система GNU/Linux — це потужна і гнучка система, яка базується на ядрі Linux та проекті GNU. Вона відрізняється від інших операційних систем відкритим кодом, що дозволяє користувачам вивчати, змінювати та розповсюджувати її.

Найбільш популярні дистрибутиви

  • Ubuntu: Найпопулярніший дистрибутив, відомий своєю простотою використання.
  • Fedora: Сучасний дистрибутив, що часто включає нові технології.
  • Debian: Відомий стабільністю, підходить для серверів.
  • Arch Linux: Орієнтований на досвідчених користувачів, забезпечує максимальну гнучкість.

Приклади команд роботи в Linux CLI

  • pwd - Показати поточну директорію
  • echo - Вивести текст на екран
  • ls - Переглянути файли в директорії
  • cd - Змінити директорію
  • touch - Створити новий файл
  • mkdir - Створити нову директорію
  • cp - Копіювати файли
  • mv - Перемістити файли
  • rm - Видалити файли
  • history - Показати історію команд

BASH-скрипт для авторизації

#!/bin/bash
echo "Введіть ім'я користувача:"
read user
echo "Введіть пароль:"
read -s pass
echo "Авторизація завершена для $user"
